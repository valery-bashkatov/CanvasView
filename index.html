CanvasView

Description

Allows you to draw curves inside the view in a convenient way.

Requirements

  • iOS 8.0+
  • Xcode 7.3+

Installation

Carthage

To integrate CanvasView into your project using Carthage, specify it in your Cartfile:

github "valery-bashkatov/CanvasView"

And then follow the instructions to install the framework.

Documentation

API Reference is located at http://valery-bashkatov.github.io/CanvasView.

Usage

import CanvasView
